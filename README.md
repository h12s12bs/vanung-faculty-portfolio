# 邱俊維教授 | 萬能科技大學企業管理系 — 個人專長介紹網站

## 📋 網站內容

本網站為邱俊維教授應徵萬能科技大學企業管理系教職之個人專長介紹網站，包含：

- **🎓 個人專長介紹** — 跨域整合的管理教育專業背景與技能
- **🤝 產學合作構想** — 五大產學合作計畫方案
- **📋 招生計畫** — 傳統招生策略與創新招生方案
- **📱 數位招生行銷** — 新社群媒體行銷策略
- **📬 聯繫方式** — 聯絡資訊與留言表單

## 🚀 技術架構

- 純 HTML5 + CSS3 + Vanilla JavaScript
- 響應式設計（RWD），支援手機、平板、桌面瀏覽
- 深色模式切換
- CSS 動畫與微互動效果
- IntersectionObserver 滾動動畫
- GitHub Pages 靜態託管
